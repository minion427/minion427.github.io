Harmony Music Studio

Project Overview

This web application is designed for a newly established local business called Harmony Music Studio, which provides private and group music lessons to students of all ages. The website’s purpose is to promote the school, provide information about available classes and events, and allow users to register for lessons online.

The intended users of this website include parents of young children, adult learners, and local musicians who want to improve their skills or join a musical community.

The website will include five primary pages: Home, About, Classes, Registration, and Events. These pages will provide users with comprehensive information about the studio’s mission, team, available classes, upcoming events, and registration options.

Page Design

  • index.html (Home)
    Purpose: Introduce the school and engage users.
    Audience: General public and prospective students.
    Content: Hero banner, welcome message, testimonial slider.
    User Input: No.
    Validations: N/A.
    Interactivity: Navigation buttons, testimonial slider.
  • about.html (About)
    Purpose: Introduce the founder and mission.
    Audience: Parents, students.
    Content: Biography, photos, school values.
    User Input: No.
    Validations: N/A.
    Interactivity: Accordion-style content.
  • classes.html (Classes)
    Purpose: Display available music lessons.
    Audience: Prospective students.
    Content: List of courses by age/instrument.
    User Input: No.
    Interactivity: Filterable course list using dropdowns and AJAX.
  • registration.html (Registration)
    Purpose: Allow students to register.
    Audience: Customers/parents.
    Content: Registration form.
    User Input: Yes.
    Validations: Name, email, instrument, preferred class.
    Interactivity: JavaScript form validation and confirmation.
  • events.html (Events)
    Purpose: Promote upcoming recitals/workshops.
    Audience: Students, parents, community members.
    Content: Event dates, descriptions.
    Interactivity: jQuery UI datepicker, collapsible event descriptions.

Dynamic Functionality

JavaScript and jQuery will be used to enhance interactivity across the site:

  • Home page: Testimonial slider (jQuery plugin).
  • About page: Accordion for staff bios (jQuery UI).
  • Classes page: AJAX filtering of class offerings by instrument/age.
  • Registration page: Form validation using JavaScript.
  • Events page: jQuery UI Datepicker to view upcoming recitals.
